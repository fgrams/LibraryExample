Library

This is a demo program written for a programming assignment, written by Frank Grams. It is a simple program for keeping track of a number of books on a set of shelves. To exit help, click "Book List" or "Shelf View".

Books can be viewed in two ways. The "Book List" view lists the entire set of books from all shelves. The "Shelf View" view lists the set of shelves, which can be selected to view the books they contain. All books must reside on a shelf. All edits to the sets of shelves and books must be made in "Shelf View". If adding shelves or books, use the "Text Entry" text box to provide a name for the new shelf or book.

To add a shelf, click "Add Shelf". To remove a shelf, select a shelf and click "Remove Shelf". Removing a shelf removes all the books it contains. To add a book, select a shelf and click "Enshelf". To remove a book, select a book and click "Unshelf".
